Workflow 2

Quote Approval

📄
hvac-quote-approval
HVAC Quote Approval Workflow

Auto-approves quotes under $1,000. Suspends and waits for manager input above.

andAgent andThen suspend resume getStepData
cost-calculation→ quote (AI)→ approval-gate→ finalize-quote

High-value quote — will suspend

Behaviour: Compressor + coil + refrigerant totals over $1,000 → workflow suspends. Resume it with the payload below.

  "ticketId": "TKT-ABC123",
  "customerName": "John Smith",
  "jobType": "replacement",
  "equipmentType": "central-ac",

Getting Started

Setup

1

Clone the repo and navigate to the example:

cd examples/with-hvac-demo
2

Create your .env file and add your Anthropic API key:

cp .env.example .env
# edit .env and set ANTHROPIC_API_KEY=sk-ant-...
3

Install dependencies and start the dev server:

pnpm install
pnpm dev
4

Open the VoltAgent console URL printed in the terminal. Paste any request payload from this page to run a workflow.

Environment Variables

ANTHROPIC_API_KEYRequiredYour Anthropic API key
VOLTAGENT_PUBLIC_KEYOptionalVoltAgent cloud console tracing
VOLTAGENT_SECRET_KEYOptionalVoltAgent cloud console tracing

Switching Models

All six agents share the MODEL constant at the top of src/index.ts:

const MODEL = "anthropic/claude-haiku-4-5-20251001"; // fast & cheap
// const MODEL = "anthropic/claude-sonnet-4-6";       // balanced
// const MODEL = "anthropic/claude-opus-4-7";          // most capable
